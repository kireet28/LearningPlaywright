# Learning Playwright

This repository contains code examples and notes for learning Playwright.

## Structure

- `Chapter_01_Basics`: Basic concepts and scripts.
- `Chapter_02_Java_Concepts`: Concepts related to Java (if applicable) or other related topics.

## Prerequisites

- Node.js
- Playwright

## Setup

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run tests using `npx playwright test`.
